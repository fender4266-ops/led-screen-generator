## ✨ Возможности

| Функция | Описание |
|---------|----------|
| 🖥️ **Мультиэкран** | Создавай композицию из нескольких экранов на одном полотне |
| 📐 **Гибкий ввод размеров** | Задавай размеры **в метрах** или **по кабинетам** |
| 🏁 **Режим «Шахматка»** | Классическая тестовая сетка для проверки пикселей |
| 🌈 **Режим «Градиент»** | Плавные переходы для проверки цветопередачи |
| 🎨 **Пресеты цветов** | 6 готовых схем + сохранение своих пресетов |
| 💾 **Автосохранение** | Свои пресеты сохраняются в браузере (localStorage) |
| 🧲 **Магнитное прилипание** | Экраны автоматически выравниваются друг к другу |
| 🖱️ **Drag & Drop** | Перетаскивай экраны мышкой прямо на холсте |
| 📤 **Экспорт в PNG** | Скачивай готовое изображение в высоком разрешении |

## 🎨 Пресеты цветов

В комплекте 6 готовых цветовых схем:

| Пресет | Цвета |
|--------|-------|
| ⚫⚪ Ч/Б | Белый / Чёрный |
| ⚫🟠 Ч/Оранж | Оранжевый / Чёрный |
| 🟢🔴 Зел/Красн | Зелёный / Красный |
| 🔵🟣 Син/Фиол | Синий / Фиолетовый |
| 🩵🌸 Циан/Роз | Циан / Розовый |
| 💛💜 Жёлт/Фиол | Жёлтый / Фиолетовый |

**Управление пресетами:**
- 🖱️ **ЛКМ** по пресету — применить к текущему экрану
- 🖱️ **ПКМ** по пресету — удалить свой пресет
- 🌐 **Применить ко ВСЕМ** — раскрасить все экраны разом
- 🎲 **Случайный** — применить случайный пресет
- ➕ **Сохранить свой** — добавить текущие цвета в список
- ↩️ **Сброс** — вернуть заводские пресеты

---

## 📖 Как пользоваться

1. **Задай размеры** экрана — в метрах или по количеству кабинетов
2. **Выбери режим** — шахматка или градиент
3. **Настрой цвета** — вручную или выбери готовый пресет
4. **Добавь экраны** — каждый новый экран создаётся с новым пресетом (можно отключить)
5. **Расставь экраны** — перетаскивай мышкой, они прилипают друг к другу
6. **Скачай PNG** — готовое изображение для загрузки на видеопроцессор

---

## 📐 Параметры по умолчанию

Каждый новый экран создаётся с настройками:

- **Размер:** 5 × 2.5 м
- **Кабинет:** 0.5 м = 192 px
- **Итоговое разрешение:** 1920 × 960 px
- **Режим ввода:** по кабинетам

## 🛠️ Технологии

- Чистый **HTML + CSS + JavaScript** (без фреймворков)
- **Canvas API** для отрисовки и экспорта
- **localStorage** для сохранения пресетов
- Работает в любом современном браузере (Chrome, Edge, Firefox, Safari)
