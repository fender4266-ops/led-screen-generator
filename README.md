# 🎨 LED Screen Generator

Генератор тестовых изображений для настройки LED-экранов.

## Возможности
- Композиция из нескольких экранов
- Режимы: шахматка, градиент
- Ввод по метрам или по кабинетам
- Пресеты цветов (с сохранением)
- Перетаскивание экранов с магнитным прилипанием
- Экспорт в PNG

## 🚀 Открыть онлайн
👉 https://fender4266-ops.github.io/led-screen-generator/

## Как пользоваться
1. Задай размеры экрана
2. Выбери режим и цвета
3. Добавь нужное количество экранов
4. Скачай PNG
